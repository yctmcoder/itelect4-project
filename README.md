# IT Elective 4 Project

## Project Description

This project is a TypeScript practice project for IT Elective 4. It demonstrates the use of TypeScript fundamentals such as primitive types, interfaces, type aliases, union types, intersection types, generic interfaces, and utility types. The project also converts JavaScript code into TypeScript with proper type annotations.

## Interfaces and Types

### Interfaces
- User
- Course
- Submission
- ApiResponse<T>

### Type Aliases
- ID
- Coordinate
- Formatter
- StringOrNumber
- Status
- StudentWithCourse

## Installation

```bash
npm install
```

## Run the project

```bash
npx ts-node src/index.ts
```
